Módulo I

Introdução ao HTML e CSS

HTML

A linguagem HTML se trata de uma linguagem desenvolvida para marcação de texto. Através do uso de tags, é possível manter um código limpo e organizado.

Uso da Div

O uso da div na linguagem de marcação HTML, ajuda na organização do código, colaborando em uma maior compreessão sobre como está organizado os blocos de código, e o que cada bloco guarda.

Tabela com principais Tag de uso comum:

Tag Descrição Exemplo de uso
<title> e </title>

Usado para definir o título da página.

<title>Título.</title>
<head> e </head>

Usado para definir um cabeçalho.

<head>Cabeçalho</head>
<hr>

Usado para demarcar uma linha horizontal que divide a página do site.

<hr>
<h1> e </h1>

Usado para demarcar títulos de páginas. Outras variações envolvem o aumento do número após a letra "h".

<h1>Exemplo</h1>
<p> e </p>

Usado para demarcar parágrafos de páginas. Parágrafos podem ser criado sem o seu uso, porém funciona como uma boa prática para legibilidade do código.

<p>Texto de exemplo.</p>
<div> e </div>

Usado para demarcar blocos de código. Usado principalmente para organizar o código e propriciar o entendimento do que colaborando bloco realiza.

<div>Bloco de exemplo.</div>
<table> e </table>

Usado para criar tabelas de conteúdo.

 <table>Tabela com conteúdos</table>
<th> e </th> Usado para definir o número de colunas de uma tabela, e nomear seu conteúdo. <th> Nome </th>
<tr> e </tr> Usado para definir uma linha dentro da tabela. <tr>Linha</tr>
<td> e </td> Usado para preencher com conteúdo, linhas e colunas da tabela. <td>Linha</td>
